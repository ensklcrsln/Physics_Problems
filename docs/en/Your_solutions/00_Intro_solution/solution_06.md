## 6. Function Analysis

Given the function:

f(x) = 3x^2 - 12x + 7

### Step 1: First derivative

f'(x) = 6x - 12

### Step 2: Find critical point

Set the derivative equal to zero:

6x - 12 = 0

x = 2

### Step 3: Second derivative

f''(x) = 6

Since f''(x) > 0, the function has a minimum at x = 2.

### Step 4: Evaluate the function

f(2) = 3(2^2) - 12(2) + 7  
f(2) = 12 - 24 + 7 = -5

### Result

Minimum point: (2, -5)
